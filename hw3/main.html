b. A new model that uses an almost flat prior for tau on the log scale

i.

model {

  for (j in 1:length(y)) {
    y[j] ~ dnorm(theta[j], 1/sigma[j]^2)
    theta[j] ~ dnorm(mu, 1/tau^2)
  }

  mu ~ dunif(-1000,1000)
  logtau ~ dunif(-100,100)
  tau <- exp(logtau)

}

vii.

  • The problem with this model is that using an imporper flat prior on \(log \tau\) implies \(p(\tau) \propto \frac{1}{\tau}\), which is itself improper. This can lead to an improper posterior, which is not a valid probability. Such a model can cause poor convergence behavior like the statistical results above.
